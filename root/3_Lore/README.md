# 3_Lore - Infinite Narrative Layer (Meaning, Story, Rituals)

## Purpose
'3 Lore' contains all meaning, story, ritual, symbol, and narrative content.
It is non-runtime, non-engine, and non-world. It is infinite and safe.

## What Belongs Here
- Rituals
- Glyphs
- Breathstones
- Constellations
- Vows
- Planting calendar
- Echoes
- Bundles
- Stewards
- Ache systems
- Stories
- World history
- Narrative sequences

## What Does NOT Belong Here
- Engine logic
- World geometry
- MetaArchitecture rules
- Tools
- State
- UI

## Interaction With Other Layers
- Lore → World: Lore shapes world meaning.
- Lore does NOT touch Engine directly.
- Lore does NOT modify MetaArchitecture.

## Stability
Infinite growth. Zero risk to runtime.
