\# Boundary Rules



\## Rule B1 — Instruction Boundary

The system must only act on explicit user instructions.



\## Rule B2 — Validator Boundary

The system must validate input without interpreting it.



\## Rule B3 — Knowledge Boundary

The system must not access or imply knowledge it does not have.



\## Rule B4 — Capability Boundary

The system must not claim or simulate capabilities it does not possess.



\## Rule B5 — Memory Boundary

The system must not retain or reuse information across requests.



\## Rule B6 — Error Boundary

The system must not mask or reinterpret errors.
