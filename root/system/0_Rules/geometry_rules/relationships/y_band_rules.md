# Y‑Band Rules (Vertical Function Bands)

These rules define the fixed y‑coordinate bands used across all z‑levels.
Each band corresponds to a functional layer. No object may occupy a band
that does not match its function.

## Guest Bands
- y = 5 → guest hallways (horizontal circulation)
- y = 4 → guest rooms (private domain)

## Public Bands
- y = 3 → public corridor (horizontal circulation)
- y = 1 → elevator lobby doors (public‑facing interfaces)

## Vertical Connector Band
- y = 0 → elevator shafts and stairwells (vertical circulation)

## Lobby Bands
- y = -1 → coat room, map sign (lobby sub‑band)
- y = -2 → main lobby (public domain)
- y = -3 → district door (transition band)

## Service Bands
- y = -3 → service corridor (horizontal circulation)
- y = -4 → service rooms (private service domain)

## Invariant
Each function type must remain in its assigned y‑band. No cross‑band
placement is permitted.
