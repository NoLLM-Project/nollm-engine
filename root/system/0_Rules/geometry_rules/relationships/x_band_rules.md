# X‑Band Rules (Horizontal Domain Separation)

These rules define the horizontal separation of domains. The x‑axis
prevents collisions between guest, public, and service regions.

## Guest Domain
- x < 0 → guest rooms and guest‑only spaces

## Circulation Spine
- x = 0 → hallways (guest horizontal circulation)

## Vertical Connectors
- x = -3 → stairwell shafts
- x = 3  → guest elevator shafts

## Public Domain
- x > 0 → public corridor and public elevator

## Invariant
Domains must remain in their assigned x‑bands. No object may cross into
another domain’s horizontal region.
