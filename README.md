# NoLLM Engine

A deterministic, rule‑based interaction engine.  
No machine learning.  
No prediction.  
No hidden layers.  
No generative components.

All behavior is explicit, inspectable, and deterministic.


---

## Source Code
This repository contains the core engine (“Hotel”) and the ecosystem scaffolding (“District”).

- **Hotel** — the deterministic runtime and rule executor  
- **District** — the future ecosystem layer (not yet open for contributions)

---

## Contributing
The core architecture is not open for modification.  
Community contributions are limited to:

- documentation  
- bug reports  
- non‑architectural improvements
